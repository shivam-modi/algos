# Advanced Algorithms and Data Structure Implementation

A comprehensive C++ collection of advanced algorithms and data structures implementations for competitive programming and educational purposes.

## 📚 Contents

### Data Structures

#### Heaps
- **maxHeap.cpp** - Max heap implementation
- **minHeap.cpp** - Min heap implementation
- **fibonacciHeap.cpp** - Fibonacci heap with advanced operations
- **minBinomialHeap.cpp** - Binomial heap implementation

#### Trees
- **splayTree.cpp** - Self-adjusting binary search tree
- **suffixTree.cpp** - Suffix tree for string processing
- **segmentTree.cpp** - Segment tree for range queries
- **diagonal-tree-traversal.cpp** - Tree traversal technique

#### Advanced Data Structures
- **maxindependentset.cpp** - Maximum independent set in trees
- **longestRouteInTree.cpp** - Finding longest paths in tree structures

### Algorithms

#### Graph Algorithms
- **fordfulkerson.cpp** - Maximum flow algorithm
- **maxbipartiteMatch.cpp** - Bipartite matching algorithm
- **vertexcover.cpp** - Vertex cover problem solution
- **dynamicConnectivity.cpp** - Union-Find with dynamic connectivity

#### String Algorithms
- **rabinkarp.cpp** - Rabin-Karp string matching algorithm

#### Optimization Problems
- **matrixChainMultiplication.cpp** - Dynamic programming for optimal matrix chain multiplication

## 🚀 Features

- **Pure C++ Implementation** - Standard library usage without external dependencies
- **Educational Focus** - Well-structured code for learning advanced algorithms
- **Optimized Solutions** - Implementations focused on time and space complexity
- **Competitive Programming Ready** - Suitable for algorithm contests and interviews
